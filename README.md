# How to develop

1. `npm install`

> 有被墙的风险，安装前建议先切换到淘宝源：<br>
> `npm config set registry 'https://registry.npm.taobao.org'`

2. `npm run dev`

> 由于Atom的terminal plus插件的问题，Mac用户目前只能在系统terminal上运行这条命令
